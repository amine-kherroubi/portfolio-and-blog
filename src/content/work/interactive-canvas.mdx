# Interactive Canvas Art

A high-performance generative art platform built with WebGL, creating real-time visual experiences that respond to user interaction and audio input.

## Project Overview

This project explores the intersection of code and visual art, using modern web technologies to create immersive, performant generative artworks that run entirely in the browser.

## Key Features

### Real-time Generation

- **60 FPS performance** using WebGL and GPU acceleration
- Dynamic particle systems with thousands of elements
- Smooth transitions and morphing effects
- Audio-reactive visualizations

### Interactive Controls

- Mouse and touch interaction
- Real-time parameter adjustment
- Multiple algorithm modes (flow fields, attractors, cellular automata)
- Export high-resolution still images and animated GIFs

### Technical Implementation

```javascript
// Fragment shader for generative patterns
const fragmentShader = `
  precision highp float;
  uniform float time;
  uniform vec2 resolution;
  uniform vec2 mouse;
  
  void main() {
    vec2 uv = gl_FragCoord.xy / resolution;
    vec2 center = mouse / resolution;
    
    float dist = length(uv - center);
    float wave = sin(dist * 20.0 - time * 2.0);
    
    vec3 color = vec3(
      0.5 + 0.5 * sin(wave),
      0.5 + 0.5 * cos(wave),
      0.5
    );
    
    gl_FragColor = vec4(color, 1.0);
  }
`;
```

## Performance Optimizations

### GPU Acceleration

All heavy computation happens on the GPU through shaders:

- Particle physics calculations
- Color transformations
- Texture sampling and blending

### Efficient Updates

```typescript
class ParticleSystem {
  private buffer: WebGLBuffer;
  private texture: WebGLTexture;

  update(deltaTime: number): void {
    // Update particle positions on GPU
    this.gl.bindBuffer(this.gl.ARRAY_BUFFER, this.buffer);
    this.gl.bufferSubData(this.gl.ARRAY_BUFFER, 0, this.particles);
  }

  render(): void {
    // Single draw call for all particles
    this.gl.drawArrays(this.gl.POINTS, 0, this.particleCount);
  }
}
```
